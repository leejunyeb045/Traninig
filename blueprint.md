
# Lotto Number Generator

## Purpose and Capabilities

This project is a simple web application that generates random lottery numbers for the user. It provides a clean and easy-to-use interface for generating a set of 6 unique random numbers between 1 and 45.

## Style, Design, and Features

### Initial Version

*   **UI:**
    *   A main title "Lotto Number Generator".
    *   A container to display the generated lottery numbers.
    *   A "Generate" button to trigger the number generation.
*   **Styling:**
    *   Centered layout.
    *   Visually distinct display for the generated numbers.
    *   Button styling for clear user interaction.
*   **Functionality:**
    *   Clicking the "Generate" button will generate 6 unique random numbers between 1 and 45.
    *   The generated numbers will be displayed in the number container.
